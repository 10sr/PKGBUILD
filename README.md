PKGBUILD
========

PKGBUILDs for Arch Linux.



Develop and Upload PKGBUILDs
-------------------

* Some prototypes are found at `/usr/share/pacman`.
* Issue `makepkg --source` to create tarball for uploading to
[AUR](https://aur.archlinux.org/)
